# _**-k parameter : used to run selected specific substring of testcases**_
eg: 
```
def test_search_add(a,b):
    if [a,b] !=  [0,0]:
        assert True
        return a+b
    else
        assert False
```
run:```pytest -k search```
This command will execute all test_search* methods in code
